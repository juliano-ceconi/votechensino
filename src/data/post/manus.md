## O que torna o Manus diferente?

Manus não é mais um chatbot genérico. É como ter um assistente pessoal que:

- **Lembra de tudo**: Contexto ilimitado (nunca mais repita “como eu disse antes…“)
- **Aprende com você**: Desenvolve personalidade baseada no seu estilo
- **Multimodal**: Entende voz, texto, até expressões faciais pela câmera

## Recursos que impressionam

### Para o dia a dia

- Agenda compromissos de forma natural (“Marca uma reunião com o time na próxima terça depois do almoço”)
- Gerencia emails e mensagens (“Responde o Pedro dizendo que amanhã mando o relatório”)
- Faz pesquisas inteligentes (“Me explica como funciona o React Server Components como se eu tivesse 5 anos”)

### Para desenvolvedores

- Debugging colaborativo (“Por que esse useEffect está rodando em loop?“)
- Geração de código com contexto (“Cria um hook customizado para fetch com TypeScript”)
- Documentação automática (“Gera docs em Markdown para essa função”)

## Exemplo de conversa real

```
Você: Manus, preciso de uma função que valide CPF em JavaScript
Manus: Já tenho uma versão aqui. Quer que eu:
1. Explique como funciona?
2. Adapte para seu projeto atual?
3. Crie testes unitários?
```

## Como começar

1. **Baixe o app** (Disponível para iOS, Android e desktop)
2. **Configure seu perfil** (Diga como quer ser chamado, seus interesses)
3. **Conecte suas contas** (Opcional: email, calendário, GitHub)
4. **Seja natural** (Fale como com um colega de trabalho)

Dica profissional: Quanto mais você usar, mais inteligente e personalizado fica. É tipo um Tamagotchi, só que útil.
